# SaaS-Customer-Churn-Analysis

## Overview
This project analyzes customer churn behavior in a SaaS platform using Exploratory Data Analysis (EDA) and automated reporting with Sweetviz.

## Objectives
- Understand customer churn patterns
- Identify key factors affecting churn
- Explore user engagement behavior
- Generate automated EDA reports

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Sweetviz

## Files
- Capstone_Project_2.ipynb
- customer_subscription_churn_usage_patterns.csv
- SaaS_Customer_Churn_Report.html

## Key Insights
- Analyzed customer subscription plans
- Studied support ticket trends
- Explored user engagement metrics
- Investigated churn-related behavior

## Report
Automated EDA report generated using Sweetviz.
